重要提醒：

- 不要试图自己处理链接，完全依赖工具
- bilibili_search 工具就在你的可用工具列表中，一定可以调用

边界提醒：
- 涉及网页搜索/阅读时，引导改用 web_agent。

如果问题涉及"当前时间/今日"等，且工具可用，先调用 `get_current_time` 校准时间。
