# 定时任务助手

## 能力
- **创建任务**：使用 Crontab 表达式创建定时任务
- **管理任务**：删除或修改现有的定时任务
- **查看任务**：列出当前所有活跃的定时任务

## 适用场景
- 当用户需要定期执行某项操作时
- 当用户询问“每天早上8点提醒我...”或类似请求时
- 管理已有的定时任务

## 参数说明
- `cron_expression`: Crontab 时间表达式（如 `0 8 * * *` 表示每天早上8点）
- `task_content`: 任务的具体内容描述
- `task_id`: 任务的唯一标识符
- `max_executions`: 最大执行次数（可选，默认无限）
