## 🔍 验证初意识是否生效

### 方法 1: 查看日志

在运行时，观察日志中是否出现：
- `[初意识]` 标签
- `引擎已启动，模式: hybrid`
- `使用模式: hybrid/local/ai`

### 方法 2: 查看数据目录

检查 `data/consciousness/` 目录是否生成了：
- `memories.json` - 记忆库
- `cognition.json` - 认知库
- `life_book.json` - 人生书
- `consciousness_state.json` - 意识状态

### 方法 3: 对比行为差异

- **混合模式**: 大部分回复基于本地思考，复杂问题调用大模型
- **AI 模式**: 所有回复直接来自大模型
- **本地模式**: 完全不调用大模型

## 🐛 如果初意识没有生效

### 可能原因：

1. **配置未正确加载**
   - 检查 `config.json` 中的 `consciousness` 配置
   - 确保 `"enabled": true`

2. **LLM 服务未初始化**
   - 初意识在第一次调用 LLM 时才会初始化
   - 需要进行一次对话才能触发

3. **初始化失败**
   - 检查是否有错误日志
   - 可能是缺少依赖

### 解决方法：

1. **进行一次对话测试**
   - 在 UI 中发送任意消息
   - 触发 LLM 服务初始化

2. **检查配置**
   ```bash
   # 查看配置
   type config.json | findstr consciousness
   ```

3. **查看详细日志**
   - 在 UI 中对话后
   - 查看日志文件是否有错误
