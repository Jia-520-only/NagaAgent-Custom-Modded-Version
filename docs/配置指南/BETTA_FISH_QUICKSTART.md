# BettaFish 融合快速开始

## 30秒快速上手

### 1. 无需配置，直接使用

启动NagaAgent后，直接在对话中使用：

```
"帮我分析一下最近AI发展的网络舆情"
"分析这句话的情感：今天天气真好"
"生成一份关于ChatGPT的舆情报告"
"全网搜索人工智能的最新动态"
```

弥娅会自动识别并调用微舆分析工具。

### 2. 启动NagaAgent

```bash
python main.py
```

启动后会看到：

```
✅ 注册MCP服务: 微舆舆情分析 (来自 mcpserver\agent_betta_fish\agent-manifest.json)
```

### 3. 测试功能

运行测试脚本验证：

```bash
python test_betta_fish.py
```

## 三种使用模式

### 基础模式（推荐新用户）

✅ 无需配置
✅ 开箱即用
✅ 使用NagaAgent的LLM

**适用场景**：
- 简单舆情查询
- 基础情感分析
- 快速生成报告

### 完整模式（推荐高级用户）

🔧 需要配置数据库
🔧 需要安装BettaFish依赖
🔧 功能最全面

**适用场景**：
- 深度舆情挖掘
- 专业数据分析
- 多源数据整合

**配置步骤**：
1. 复制 `betta-fish-main/.env.example` 为 `.env`
2. 填写数据库配置
3. 安装依赖：`cd betta-fish-main && pip install -r requirements.txt`

### 混合模式（推荐生产环境）

⚡ 自动智能切换
⚡ 简单任务用基础模式
⚡ 复杂任务用完整模式

## 常见问题

### Q: 需要额外安装软件吗？

**A**: 基础功能不需要。完整功能需要MySQL数据库。

### Q: 会增加系统负担吗？

**A**: 基础模式几乎零负担，完整模式会增加一些资源消耗。

### Q: 能否使用其他LLM？

**A**: 可以。在 `betta-fish-main/.env` 中配置独立的API密钥。

### Q: 如何提升分析效果？

**A**:
1. 配置完整模式（使用BettaFish的深度分析）
2. 配置MySQL数据库（积累历史数据）
3. 使用专业模型（如多语言情感分析）
