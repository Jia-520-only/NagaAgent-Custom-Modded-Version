## 📊 功能概览

### 核心功能
- ✅ 多端对话（PC、网页、移动端）
- ✅ QQ/WeChat 机器人集成
- ✅ TTS 语音合成
- ✅ AI 绘图
- ✅ 记忆系统（GRAG）
- ✅ 工具调用（MCP）
- ✅ 舆情分析（BettaFish）

### 高级功能
- ✅ Live2D 虚拟形象
- ✅ 实时语音交互
- ✅ 物联网集成（MQTT）
- ✅ 多智能体协作
