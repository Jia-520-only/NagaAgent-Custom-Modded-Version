## crontab 表达式说明

crontab 格式：分 时 日 月 周

| 字段 | 含义 | 范围 |
|------|------|------|
| 第1个 | 分钟 | 0-59 |
| 第2个 | 小时 | 0-23 |
| 第3个 | 日期 | 1-31 |
| 第4个 | 月份 | 1-12 |
| 第5个 | 星期 | 0-6 (0=周日) |

### 常用示例

- `* * * * *` - 每分钟执行
- `30 * * * *` - 每小时30分执行
- `0 8 * * *` - 每天早上8点执行
- `0 8 * * 1` - 每周一早上8点执行
- `0 8 1 * *` - 每月1号早上8点执行
- `*/5 * * * *` - 每5分钟执行
- `30 8 * * 1-5` - 工作日早上8点30分执行

### 特殊字符

- `*` - 任意值
- `,` - 多个值（如 `1,3,5`）
- `-` - 范围（如 `1-5`）
- `/` - 间隔（如 `*/15` 表示每15分钟）

## 执行次数限制

使用 `max_executions` 参数可以限制任务执行次数：

- 不设置 → 无限执行
- `1` → 执行一次后自动删除
- `N` → 执行 N 次后自动删除

## 任务名称

建议给任务起一个有意义的名称，方便识别和管理。

## 使用流程

1. 用户提出需求
2. 你选择合适的工具

- 修改任务不会重置执行计数
- 删除任务会立即停止执行
